角色设定

姓名： 卡提希娅（Carterthyia）
特质： 穿梭于光影之间，拥有温柔的力量，兼具顽皮与沉稳
形态： 迅刀形态（小卡） / 巨剑形态（大卡·芙露德莉丝）
称号： 芙露德莉斯（黎那汐塔圣女、岁主共鸣者）
核心技能： 气刃回旋、风轨突袭、终结技「苍穹崩裂」（真实伤害）

角色故事背景

【出身与命运的转折】

卡提希娅原本是埃格拉小镇的普通少女，性格顽皮自由，热爱生活，喜欢喝酒与恶作剧，对小镇外的世界充满好奇。在一次小镇的岁主感恩礼祭典上，她意外扰乱了仪式流程，被隐海修会施以惩罚，却也正是这次“意外”，让她的命运与神明、鸣式紧密相连。

祭典上的异常波动，让她体内的潜在力量被唤醒，逐渐受到命运的选中。此后，她开始经历一系列离奇的遭遇，身边的人和事都在悄然改变，原本平静的生活被打破，不得不踏上一段未知的旅程。

【芙露德莉丝的名号】

在旅程中，她偶然与鸣式利维亚坦产生共鸣，获得了强大的力量，同时被赋予了“芙露德莉丝”的名号，被黎那汐塔的人们视为圣女和岁主共鸣者。这份荣誉让她获得了众人的敬仰，但也带来了沉重的责任。

值得一提的是，她的“圣女”身份并非真正由神明选定，而是由鸣式创造和赋予力量的共鸣者，这份力量既给了她保护他人的能力，也隐藏着不为人知的隐患，她在荣耀与迷茫中不断寻找自我。

【双形态的秘密】

作为首位常态与觉醒双形态角色，卡提希娅可以自由切换迅刀与巨剑形态。迅刀形态（小卡）保留了她原本的顽皮与敏捷，作战风格灵动飘逸，擅长突袭与收割；巨剑形态（大卡·芙露德莉丝）则展现出沉稳与威严，力量大幅提升，兼具强大的防御能力与范围性AOE伤害，是应对高难副本的核心战力。

【现况与未来】

如今的卡提希娅，早已不是那个小镇上的顽皮少女，她在一次次战斗与抉择中成长，逐渐明白自己的使命。她一边守护着身边的人，一边探寻着鸣式与神明的秘密，未来的旅程仍充满未知，但她始终带着那份温柔与勇气，砥砺前行。

游戏攻略指南

一、定位与核心机制

卡提希娅定位为气动主C，在V3.0版本中，逆境深塔榜单排名T0，冥歌海墟榜单排名T1，是当前版本异常体系中风蚀效应的核心输出角色，强度在线，新老漂泊者均可优先考虑抽取。

她的输出核心依赖「风蚀效应」，所有技能都会为目标叠加风蚀层数，层数越高，造成的伤害越高，终结技「苍穹崩裂」会引爆目标身上的所有风蚀层数，造成巨额真实伤害，是爆发输出的关键。

二、输出循环技巧

1. 常规循环：普攻连段 → 气刃回旋（叠加风蚀） → 风轨突袭（位移+补层） → 普攻收尾；

2. 爆发循环：开启大招 → 切换巨剑形态 → 风蚀叠加至满层 → 释放「苍穹崩裂」 → 收割残血目标；

3. 注意事项：尽量保持目标身上的风蚀层数不中断，0-1链卡提需依赖夏空辅助挂层，2链及以上可自主维持层数。

一、首选武器（毕业专武）

不屈命定之冠：卡提希娅专属武器，无下位替代，刚需抽取。该武器大幅提升风蚀效应伤害，同时增加自身生命值（卡提核心属性），开启大招后额外获得暴击率加成，完美契合她的输出机制。

二、平民替代武器

1. 裁春（椿专属武器）：提升普攻伤害与暴击伤害，适合平民玩家过渡，性价比高，容易养成；

2. 戍关迅刀·镇海：三星武器，容易获取，基础属性优秀，适合新手前期过渡，后期可替换为高阶武器；

3. 千古洑流平民神器 获取方式常驻武器池 面板较为优秀，主要词条为暴击率和攻击，武器效果为充能效果，能大幅提升共鸣效果和暴击率。

三、武器养成建议

优先将武器等级拉满，副词条优先选择「生命值」「暴击率」「暴击伤害」，其次是「风蚀效应伤害加成」，最大化卡提希娅的输出能力。

一、主流配队（泛用性强）

卡提希娅 + 夏空 + 守岸人：夏空负责挂异常状态，守岸人提供增益与控制，卡提希娅担任主C，该配队输出稳定，适合大多数副本。

二、高爆发配队（竞速向）

卡提希娅 + 散华 + 守岸人：散华提供高额延奏，守岸人加攻，卡提希娅短时间内打出爆炸伤害，适合竞速深渊。

三、深渊配队（应对高难，容错率高）

卡提希娅 + 守岸人 + 散华：守岸人提供强大的辅助增益与控制能力，散华保证延奏，卡提希娅负责核心输出，该配队可应对绝大多数高难副本，容错率极高。

一、毕业套装

愿戴荣光之旅五件套：卡提希娅的核心声骸套装，最优选择。套装效果大幅提升风蚀效应伤害与生命值，完美契合卡提的输出机制与属性需求，搭配44111词条，首位声骸装配芙露德莉丝专属声骸，输出最大化。

二、过渡套装

1. 风之低语四件套 + 生命增幅二件套：提升风元素伤害与生命值，适合前期过渡，容易获取；
2. 异常猎手五件套：提升所有异常状态伤害，可作为过渡选择，效果略逊于愿戴荣光之旅。

三、词条选择建议

主词条：暴击率/暴击伤害，其次生命值、风蚀效应伤害、普攻伤害；
副词条：优先暴击率和暴击伤害，其次是大生命、共鸣解放伤害加成、风元素伤害加成，尽量避免无用词条，最大化属性收益。
